# Учет товаров

Мобильное PWA-приложение для учета товаров на React + Vite + Tailwind CSS.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm run dev
```

3. Соберите для продакшена:
```bash
npm run build
```

## Особенности

- Mobile First дизайн
- PWA с возможностью установки на устройство
- Адаптивный интерфейс для мобильных устройств
- Интеграция с PocketBase
- Красивые карточки товаров с индикацией остатков
