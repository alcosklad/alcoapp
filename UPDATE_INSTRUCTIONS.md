# Обновление категорий и объемов товаров

## Что делает скрипт
Скрипт автоматически:
1. Определяет категорию товара по названию (водка, виски, вино и т.д.)
2. Извлекает объем из названия (0.7л, 700мл и т.д.)
3. Обновляет товары в PocketBase

## Как запустить

1. Открой терминал в папке проекта:
```bash
cd /Users/rostislavkomkov/Desktop/alcoapp
```

2. Запусти скрипт:
```bash
node update-products.js
```

## Что будет сделано
- Товары без категории получат категорию на основе названия
- Товары без объема получат объем из названия
- Остальные поля не изменятся

## Категории, которые определяются:
- Водка
- Виски
- Вино
- Коньяк
- Ром
- Текила
- Джин
- Ликер
- Шампанское
- Пиво
- Другое

## Форматы объема, которые распознаются:
- 0.5л, 0.7л, 1л, 1.5л
- 0.5 л, 0.7 л (с пробелом)
- 500мл, 700мл, 1000мл
- 500 мл, 700 мл (с пробелом)

После запуска скрипта все товары будут иметь категории и объемы, и фильтры в PriceList заработают корректно!
